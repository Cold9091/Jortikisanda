# JortikiSanda - Website Institucional

![JortikiSanda Logo](https://img.shields.io/badge/JortikiSanda-Contabilidade%20Angola-blue?style=for-the-badge)

## 📋 Sobre o Projeto

Website institucional completo para a **JortikiSanda**, empresa angolana especializada em serviços contábeis, fiscais e de gestão empresarial. O projeto foi desenvolvido com foco na experiência do usuário, design responsivo e funcionalidades modernas para facilitar o contato e apresentação dos serviços.

## 🌟 Características Principais

### ✨ Páginas do Website
- **Página Inicial** - Apresentação da empresa com navegação para todas as seções
- **Sobre Nós** - História, valores e diferenciais da empresa
- **Serviços** - Detalhamento completo dos serviços oferecidos
- **Planos** - Três planos de serviços (Básico, Médio, Avançado) com comparação detalhada
- **Blog** - Artigos e insights sobre contabilidade e gestão empresarial em Angola
- **Depoimentos** - Feedback detalhado de clientes satisfeitos
- **FAQ** - Perguntas frequentes organizadas por categorias
- **Contato** - Múltiplas formas de contato e formulário integrado

### 🎨 Design e UX
- Design responsivo para desktop, tablet e mobile
- Animações suaves e micro-interações
- Paleta de cores profissional (azul, dourado, branco)
- Tipografia moderna (Montserrat)
- Efeitos glassmorphism e gradientes
- Indicadores visuais de prioridade no calendário fiscal

### 📱 Funcionalidades
- **Formulário de Contato** que redireciona para WhatsApp
- **Navegação por páginas** com sistema de roteamento
- **Blog com Sistema de Busca** e categorização de artigos
- **Sistema de Planos** com comparação detalhada
- **Design Responsivo** completo
- **Otimização SEO** básica implementada
- **Páginas de Leitura** dedicadas para cada artigo do blog

## 🛠️ Tecnologias Utilizadas

### Frontend
- **React 18** - Biblioteca principal
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Framework de estilização
- **Vite** - Build tool e dev server
- **Wouter** - Roteamento leve
- **Lucide React** - Ícones modernos

### UI Components
- **Radix UI** - Componentes acessíveis
- **Shadcn/ui** - Sistema de componentes
- **React Hook Form** - Gerenciamento de formulários
- **TanStack Query** - Gerenciamento de estado server

### Backend
- **Express.js** - Servidor web
- **TypeScript** - Tipagem no backend
- **Drizzle ORM** - ORM type-safe
- **Zod** - Validação de schemas
- **Express Session** - Gerenciamento de sessões

## 🚀 Como Executar o Projeto

### Pré-requisitos
- Node.js 18+ 
- npm ou yarn

### Instalação
```bash
# Clonar o repositório
git clone [url-do-repositorio]

# Entrar no diretório
cd jortikisanda-website

# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev
```

### Scripts Disponíveis
```bash
npm run dev          # Servidor de desenvolvimento
npm run build        # Build para produção
npm run preview      # Preview do build
npm run type-check   # Verificação de tipos
```
